# Banco de Mídias

A console app to manage a personal media catalog built with Java.

## About
I built this project to practice OOP concepts in Java, like inheritance, polymorphism, 
encapsulation, and composition. It started as a simple console app and evolved into a 
more structured project, with proper package organization, service layers, and custom exception handling. 
This project will continue to evolve along with my learning and personal development.
 

## Features

- Add books, music and podcasts
- List all media
- Search by title
- Mark as read/listened and unread/not listened
- Custom exception handling for invalid inputs and missing media

## Technologies

- Java 21
- OOP (Inheritance, Polymorphism, Encapsulation, Composition)
- Custom exceptions
- Package structure with model, service and application layers

## How to run

1. Clone the repository
2. Open in Eclipse or any Java IDE
3. Run Main.java inside bancoDeMidias.application
